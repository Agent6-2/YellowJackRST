Configuration du Système de Semaines

Cliquez sur le bouton ci-dessous pour initialiser le système de gestion des semaines dans la base de données.

Ce script va :
  • Créer la table weeks
  • Ajouter la colonne week_id aux tables existantes
  • Créer la première semaine si aucune n'existe
  • Assigner les données existantes à la première semaine
Initialiser le Système de Semaines

Note : Cette opération ne doit être exécutée qu'une seule fois. Si le système est déjà configuré, vous pouvez accéder directement au panneau de gestion des semaines.
